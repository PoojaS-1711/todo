# Todo

# To-Do List Application

## Overview

The To-Do List Application is a modern task management web application built using HTML, CSS, and JavaScript. It allows users to efficiently organize, track, and manage their daily activities through an intuitive and responsive interface.

The application demonstrates core front-end development concepts including DOM manipulation, event handling, state management, local storage persistence, dynamic content rendering, and responsive design.

---

## Features

### Task Management (CRUD Operations)

* Create new tasks
* View existing tasks
* Edit task details
* Delete tasks

### Task Categories

Organize tasks into meaningful categories:

* Daily Must-Dos
* Professional & Work
* Chores & Errands
* Health & Wellness
* Personal Goals & Learning
* Long-Term / Someday

### Task Status Management

* Mark tasks as completed
* Toggle completed tasks back to active status
* Visual distinction between active and completed tasks

### Advanced Filtering

* View All Tasks
* View Active Tasks
* View Completed Tasks

### Data Persistence

* Uses `window.localStorage`
* Tasks remain saved even after refreshing or closing the browser
* Automatic saving on every modification

### Responsive Design

* Mobile-friendly layout
* Tablet compatibility
* Desktop optimization
* Flexible interface using modern CSS

### Modern UI Features

* Animated gradient background
* Glassmorphism design effects
* Hover animations
* Interactive buttons
* Smooth transitions
* Dynamic task cards

## How It Works

### Adding Tasks

1. Enter a task in the input field.
2. Select a category.
3. Click the "Add Task" button.
4. The task is saved and displayed instantly.

### Editing Tasks

1. Click the Edit button.
2. Update the task text.
3. Save changes.

### Deleting Tasks

1. Click the Delete button.
2. The task is removed permanently.

### Completing Tasks

1. Check the task checkbox.
2. The task is marked as completed.
3. Completed tasks receive visual styling.

### Filtering Tasks

Use the filter buttons to switch between:

* All Tasks
* Active Tasks
* Completed Tasks
